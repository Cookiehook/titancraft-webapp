Welcome TitanCrafters!

Welcome to your one-stop-shop for information about the Titancraft server. Please read the instructions below in order to use this site.

This site is run by Cookiehook, with assistance from the lovely Titancraft community. Please do not ask the Titancraft mods for support, they will not be able to help. If you think a record for a shop/farm is wrong, please contact the shop/farm staff. If you are having difficulty using this site, please contact Cookiehook on Discord.

To create an account, click the "Log In" button in the top-right corner of this site. This will take you to the Discord authorisation page, where you will be asked to Authorise the titancraft-shop app. This allows the site to check that you are part of the Titancraft server before allowing access. Click Authorise to continue. Access can be revoked at any time by going to accessing Settings -> Authorised Apps in Discord.

You will not be able to access any other parts of the site until you have created an account.
